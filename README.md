# FORMACION-R-Y-R-STUDIO
El objetivo de esta formación es introducir a los participantes en el uso y manejo de RStudio, un entorno de desarrollo integrado (IDE) para el lenguaje de programación estadístico R.
R es una herramienta ampliamente utilizada por estadísticos, economistas, científicos de datos, investigadores y analistas de diversas disciplinas, gracias a su potencia, flexibilidad y su enorme comunidad de usuarios.

A lo largo de este curso, los asistentes aprenderán a:

Importar y gestionar conjuntos de datos desde múltiples fuentes (archivos CSV, Excel, bases de datos, etc.).

Limpiar y transformar datos para su análisis.

Realizar análisis estadísticos descriptivos e inferenciales.

Crear visualizaciones de datos estáticas y dinámicas de alta calidad.

Generar informes reproducibles que integren código, resultados y explicaciones.

El curso está diseñado tanto para principiantes absolutos como para aquellos que deseen afianzar sus conocimientos, haciendo hincapié en la práctica constante y en la resolución de problemas reales. Al finalizar, los participantes contarán con las competencias básicas para abordar proyectos de análisis de datos con RStudio de manera autónoma y eficiente.
